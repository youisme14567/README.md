## problem description
UniWell is a application designed for university students to track their health habits including water intake, sleep, exercise and meals. This app supports SDG3 by promoting Good Health and Well-Being through this app, it helps students build healthier daily rountine.

## SDLC Overveiw
We planned the application features and requirements based on SDG3, designed wireframes and full mockup in figma all screen flows and buttons int the prototype, and evaluated the design to ensure it meets the project requiremnts
